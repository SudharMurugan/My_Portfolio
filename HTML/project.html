Lung Cancer Detector

Description: Designed a Flask-based web application leveraging a deep learning model to detect lung cancer from CT scans. The application aims to assist in early diagnosis and improve patient outcomes.

  • Utilized the Efficient B0 CNN model for accurate predictions.
  • Integrated data visualization for detailed analysis of prediction results.
  • Developed an intuitive interface for users to upload and analyze images.

Tools Used: Python, Flask, TensorFlow, OpenCV

Outcome: Enabled healthcare professionals to perform preliminary screenings, potentially saving lives through early detection.

Project Details: This project uses deep learning to automate the detection of lung cancer, reducing diagnostic time and increasing accessibility for remote healthcare. It integrates seamlessly with a user-friendly interface for image uploads and analysis.
